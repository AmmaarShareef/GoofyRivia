# GoofyRivia

**GoofyRivia** is a local **Jackbox‑style murder trivia game** built with TypeScript, React (frontend), and Node.js/Express (backend). Players can join a lobby, receive questions, and play in real time — all locally on your machine.

## Features

- Generates a random room code to start a game
- Frontend in React to display lobby and game screens
- Backend API in Express/TypeScript to serve game logic
- Easy local setup and play
- Extensible design for future multiplayer and realtime enhancements

## Tech Stack

- **Frontend:** React + TypeScript (Vite)
- **Backend:** Node.js + Express + TypeScript
- **Communication:** REST endpoints (future WebSocket support)
- **Tooling:** ts‑node for backend dev
